/* === 现代主题 Modern Theme === */
/*
背景色彩：
- 主背景: 蓝白渐变 (清新而不刺眼)
- 按钮: 浅灰到白色渐变
- 悬停: 淡蓝色效果
- 文字: 深灰色 (易读性佳)

视觉特点：
✓ 现代扁平化设计
✓ 柔和的圆角边框
✓ 优雅的颜色过渡
✓ 专业而温和的外观
✓ 适合长时间使用
*/

/* === 深色主题 Dark Theme === */
/*
背景色彩：
- 主背景: 深灰色调
- 按钮: 深色背景
- 悬停: 较亮的深色
- 文字: 浅色 (在深色背景上)

视觉特点：
✓ 护眼的深色调
✓ 适合夜间使用
✓ 减少屏幕眩光
✓ 现代化深色美学
✓ 节省电池电量(OLED屏幕)
*/

/* === 使用建议 Usage Recommendations === */
/*
现代主题适用场景：
- 白天办公环境
- 明亮的工作空间
- 需要专业外观的场合
- 长时间文档处理工作

深色主题适用场景：
- 夜间工作
- 昏暗环境
- 眼部敏感用户
- 追求酷炫外观
*/
